# DataFest-Hackathon-2025-Loyala


## 📌 Project Overview
This repository contains the work submitted by our team for the **ASA DataFest Hackathon 2025** held at Loyola University Chicago. The project focuses on analyzing large-scale datasets to identify trends, build predictive models, and provide data-driven recommendations.

## 🚀 Key Features
- **Exploratory Data Analysis (EDA):** Deep dive into the dataset using Python to identify patterns and anomalies.
- **Machine Learning Models:** Implementation of predictive algorithms to forecast outcomes.
- **Data Visualization:** Comprehensive charts and graphs to make complex data accessible.
- **Presentation:** The final slide deck summarizing our methodology and findings.

## 📁 Repository Structure
- `DataFeast_Loyola.ipynb`: The primary Jupyter Notebook containing all data cleaning, analysis, and modeling code.
- `svh_datafest_final1.pptx`: The final presentation deck used for the hackathon judging.
- `README.md`: Project documentation and setup guide.

## 🛠️ Tech Stack
- **Languages:** Python (Jupyter Notebook)
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
- **Tools:** GitHub, Microsoft PowerPoint
